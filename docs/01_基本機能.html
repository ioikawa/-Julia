基本機能

Juliaの起動

ここでは全学計算機システムの Ubuntuサーバに sshでログインし，Juliaを起動する．

macOSやUbuntu，あるいは Windows Subsystem for Linuxのターミナルから，ssh コマンドでリモートログインする．

$ ssh 全学計算機システムのユーザー名@ubuntu.u.tsukuba.ac.jp
# パスワードを入力

julia を実行するとJuliaが起動し，次のように表示される．

$ julia
               _
   _       _ _(_)_     |  Documentation: https://docs.julialang.org
  (_)     | (_) (_)    |
   _ _   _| |_  __ _   |  Type "?" for help, "]?" for Pkg help.
  | | | | | | |/ _` |  |
  | | |_| | | | (_| |  |  Version 1.7.2 (2022-02-06)
 _/ |\__'_|_|_|\__'_|  |  https://github.com/conda-forge/julia-feedstock
|__/                   |

julia>

これは REPL[repl] モードと呼ばれるもので，プログラムの入力と実行結果の表示を対話的に行うことができる．

Note

コードをファイル file.jl に保存すれば，ターミナルから $ julia file.jl で実行できる．あるいは，Juliaを起動後，julia> include("file.jl") で読み込んで実行できます． 前者は実行時に毎回コンパイル（コードから実行形式を生成すること）するので，時間がかかることがあるので注意．

四則演算

試しに四則演算で計算してみよう．

julia> 1 + 2    # 足し算3
julia> 1 - 2    # 引き算-1
julia> 2 * 3    # 掛け算6
julia> 2 / 3    # 割り算0.6666666666666666
julia> 10^3     # 累乗． Pythonでは 10**3 と記す．1000
レポート作成上の注意

最後の演算結果は ans で参照できますが， Jupyter Notebookではセルの実行順序によって，ans が予期しない値になる可能性があります． レポートでは計算結果を ans で参照しないで，きちんと変数に格納ください．

コメントアウト

シャープ記号 # 以降のコードは行末までコメントアウト（＝無視）される． 行の中で部分的に，あるいは複数行をコメントアウトしたい場合は，#= ... =#で囲むこともできる．． 

julia> begin
       println("1行目")
       # println("2行目：ここは実行されない")
       end1行目

begin ... end で囲うとコードのひとまとりのグループとして扱うことができる． 本講義では主にREPLモードで出力を抑制するために用いる．

Tip

レポートではコメントアウト機能をプログラムの説明を記述するのに積極的に使ってください．

文字列: Strings

文字列を表示する簡単な例として，println()を使って hello, world [HW]と表示してみよう． 文字列は" ... "で囲む必要がある．

julia> println("hello, world")hello, world

println()は自動的に末尾に改行コードが入るが，print()は入らない．

julia> print("hello")       # print()は改行なしhello
julia> print(", world"), world
julia> println("hello, \nworld")   # `\n`は改行コードhello, 
world

複数行の文字列は，""" ... """ で囲む．Python でも同様である．

julia> println(
       """
             _       _ _
            | |_   _| (_) __ _
         _  | | | | | | |/ _` |
        | |_| | |_| | | | (_| |
         \\___/ \\__,_|_|_|\\__,_|
       """
       )                                    
      _       _ _       
     | |_   _| (_) __ _ 
  _  | | | | | | |/ _` |
 | |_| | |_| | | | (_| |
  \___/ \__,_|_|_|\__,_|

文字列の連結は * で行う．

julia> println("hello" * ", world")hello, world
julia> println("Hello! "^5)Hello! Hello! Hello! Hello! Hello!
julia> println("1"^19)   # 1のrepunit1111111111111111111

ちなみに，1を2, 19, 23, 317, 1031個の並べた数は素数である．[repunit]

Note

Pythonなどの他の言語では， 乗算記号*ではなく + で文字列の連結を行います． 公式ドキュメントによると，数学において，+ は可換な演算を表すので，Juliaでは， 非可換な演算にも用いられる * を文字列の連結に使うようにしたらしい．
