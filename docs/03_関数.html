関数: Functions

関数の記述

Julia言語における関数 (function) とは，与えられた引数 (argument)をもとに， 一連の処理を実行し，値を返却するオブジェクトである． 関数は function ... end で定義する．

例として，２つの引数 x, y に対して，和 x+y を表示して，かつ和を返却する関数を定義する．

julia> function f(x,y)
           x + y
       endf (generic function with 1 method)
julia> f(1,2)3

これは，数学の関数と同様に形式で書くこともできる．

julia> g(x,y) =  x + yg (generic function with 1 method)
julia> g(1,2)3

次のように複数の処理を含む関数は，function ... end を用いたほうがわかりやすい．

julia> function h(x,y)
           z = x + y
           println("$x + $y = $z")    # 文字列中で`$x`と書くと，変数`x`に置き換わる．
       endh (generic function with 1 method)
julia> h(1,2)1 + 2 = 3

引数は必ずしも必要ない．

julia> function sayhello()
           println("Hello!")
       endsayhello (generic function with 1 method)

戻り値の指定 return

関数の戻り値は return で明示できる． return に到達した時点で関数から抜ける． 

julia> function mult(x,y)
           println("ここは実行される")
           return x*y
           println("return後のコードは実行されない")
       endmult (generic function with 1 method)
julia> mult(2,3)ここは実行される
6

単にreturnと書くと，nothingが返却される．

function test()
    println("ここは実行される")
    return
    println("return後のコードは実行されない")
end
@show test()
ここは実行される
test() = nothing

オプション引数 (optional argument)

引数において，引数名=値と記述するとデフォルト値を指定できる．

julia> F(x, y=100) = x + yF (generic function with 2 methods)
julia> F(1,2)3
julia> F(1) # = F(1,100)101
julia> G(x=1, y) = x+y  # Error! optional argumentは後方のみで利用できるERROR: syntax: optional positional arguments must occur at end around REPL[4]:1

キーワード引数 (keyword arugment)

引数をセミコロン;で区切った場合，; 以降は「キーワード引数」というものになる． キーワード引数の順序は自由であるが，キーワード引数は必ず 引数名=値の形で指定する必要がある． 

julia> H(x; a, b) = a*x + bH (generic function with 1 method)
julia> H(1,2,3) # これはエラーになるERROR: MethodError: no method matching H(::Int64, ::Int64, ::Int64)
Closest candidates are:
  H(::Any; a, b) at REPL[1]:1
julia> H(1, a=2, b=3)5
julia> H(1, b=3, a=2)5
julia> H(a=2, b=3, 1)5
julia> H(a=2, 1, b=3)5
Danger

キーワード引数と通常の引数の順番を入れ替えて混ぜることもできますが， 読みにくくなるので本講義では非推奨とします．
