はじめに

Julia言語とは

Julia言語は数学計算に関するコードの「書きやすさ」と「実行速度」を両立させた数値計算向けに設計された汎用言語である．

  • 比較的新しい言語で，2018年にバージョン 1.0 がリリースされ[1]，言語仕様が固まった．様々な先発言語の良い点や現代的な機能を取り入れている．
  • 文法・構文に関してはMATLABから大きな影響を受けている．
  • 数学関数，有理数，複素数，任意精度計算，線形代数，などなどの多数の機能が標準で実装されている．
  • Pythonと同様に動的型付けなので，コードが簡潔に書きやすい．一般に動的型付け言語は実行速度が遅いというデメリットがあるが，LLVMという中間言語や Just-In-Time (JIT) コンパイルなどの技術を活用することで，その欠点をカバーしている．実行速度は，プログラミング言語の中で最速と言われるC言語と比べてもおよそ数分の1と評価されている[2]．

実行環境

  • 全学計算機システムではJuliaがインストール済みなので，端末から利用可能である．UbuntuサーバーにSSH接続すればネットワーク越しでも利用できる．→ 本講義ではこれを基準とする．
  • ローカル（自分のPC）に実行環境を構築することも簡単である．ただし，本講義ではサポートできない．インストールの失敗や実行時のエラーは，環境に依存するので解決が難しいことがある． Jupyter Notebookも自力でインストールできる人には便利かもしれない．→参考
  • CoCalc: Jupyter Notebook 上 Juliaを実行できるクラウドサービス．無料アカウントでも充分に使える．こちらも講義ではサポートはできない．
