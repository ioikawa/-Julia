条件分岐

変数の値によって処理を変えたい場合， if ... [elseif ... else ...] end を用いる．

例えば，変数 x が x>0 であるときに "positive" と表示するには， 次のように記述する．

x = 1
if x>0
    println("$x is positive.")
end
1 is positive.

x < 0 のとき "negative" と表示する場合は，elseif で条件式を追加すればよい．

x = -2
if x > 0
    println("$x is positive.")
elseif x < 0                      # 追加
    println("$x is negative.")    # 追加
end
-2 is negative.

x<0 でも x>0 でもないときは，x は 0 と言いたいところだが，それ以外の可能性もあるので， else を用いて次のように書いておこう．

x = NaN
if x > 0
    println("$x is positive.")
elseif x < 0
    println("$x is negative.")
else                                      # 追加
    println("$x is zero or something.")   # 追加
end
NaN is zero or something.

数値に関する等号と比較演算子

数値どうしを比較する演算子は次の通り．

プログラム条件式
x == y$x=y$
x != y$x\neq y$
x > y$x > y$
x >= y$x \ge y$

論理積（and）は &&，論理和 (or) は || が対応している．例えば，

julia> 1 < 2 && 2 < 3true

これは次のように書くこともできる．

julia> 1 < 2 < 3true

短絡評価

x=1
if x > 0
    println("$x is positive.")
end
1 is positive.

は && を用いて次のように書くことも多いようである．

x=1
x>0 && println("$x is positive.")
1 is positive.

最初に x>0 の評価が行われ， true ならばもう片方の評価が行われる．つまり，println 部分が実行される． もし，false ならば，残りの条件式の評価はスキップされる．
